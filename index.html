Notes on metabolic modelling analysis

a how-to with some basic background and applications

By Haris Zafeiropoulos

Notes on metabolic modelling analysis uses examples and invokes several tools and libraries to explain the ideas behind various types of metabolic modelling analysis! Some basic background on genome scale metabolic reconstructions might be expected.

Notes on metabolic modelling analysis is executable! Each chapter is actually generated from Python code. If you view the HTML version of the book, you can click on the "Launch Binder" icon at the top of each page to start an interactive version of that chapter.
